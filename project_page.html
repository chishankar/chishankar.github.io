CMSC131: Introduction to Object Oriented Programming

Fall 2016

Project 1: CrypoQuiz & Multilingual Greetings

Language: Java

This project was designed as an introduction to Java. It was meant to introduce the concept of user input, if/else statements, booleans, primitive types.

Project 2: CryptoCode

Language: Java

This project was designed to introduce the concepts of string manipulation and casting.

Project 3: Flag Maker

Language: Java

This project involved us creating colored flags that represented country flags. The point of this project was to learn and practice the function single and nested for loops, while loops, using methods, understanding objects/classes and user input.

Project 4: Photo Tools

Language: Java

This project involved us creating a photo editor.The point of this project was to understand using parameters, methods, for loops and objects/classes. We were required to implement the following:
  • Making a copy of the photo
  • Blue filter
  • Grayscale effect
  • Artsy effect
  • Censoring effect
  • Stretched effect
  • Mirror effect
  • Rotation

Project 5: Martian Cubic Calculator

Language: Java

This project involved us creating a polynomial calculator that calculated the basic calculations. The concept of this project was to get comfortable with modifiers (public, static, final, etc...), method overloading, string manipulation, 'this' keyword to current objects, and object manipulationand creating student tests to test our code.

Project 6: Poker

Language: Java

This project involved us creating a hand evaluator/poker project to evaluate cards and correctly identify the type of hand and ranking. This project was designed to introduce us to array data structures, objects, creating methods, writing student tests, and object manipulation.

Project 7: Battleable

Language: Java

This project involved us creating objects or objects with characteristics that battle each other. This project was designed for us to understand the concept of creating a class, methods, data structures, enums, and past concepts from preivous projects.
