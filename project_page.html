CMSC131: Introduction to Object Oriented Programming

Fall 2016

Project 1: CrypoQuiz & Multilingual Greetings

Language: Java

This project was designed as an introduction to Java. It was meant to introduce the concept of user input, if/else statements, booleans, primitive types.

Project 2: CryptoCode

Language: Java

This project was designed to introduce the concepts of string manipulation and casting.

Project 3: Flag Maker

Language: Java

This project involved us creating colored flags that represented country flags. The point of this project was to learn and practice the function single and nested for loops, while loops, using methods, understanding objects (how classes work) and user input.

Project 4: Photo Tools

Language: Java

This project involved us creating a photo editor that allowed us to do the following:
  • Making a copy of the photo
  • Blue filter
  • Grayscale effect
  • Artsy effect
  • Censoring effect
  • Stretched effect
  • Mirror effect
  • Rotation
